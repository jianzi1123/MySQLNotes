

1.查询员工12个月的工资总和，并起别名为ANNUAL SALARY

```mysql
# 基本工资
SELECT employee_id, last_name, salary * 12 "ANNUAL SALARY"
FROM employees;

# 基本工资+奖金
SELECT employee_id, last_name, salary * 12 * IFNULL(commission_pct, 0) "ANNUAL SALARY"
FROM employees;
```
